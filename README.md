#oyyd blog

### This version

__Features:__

- ES6 + async await + SourceMap
- Flexibility > Performance
- HMR
- Server Side Rendering

__Exclude:__

- Heavy Usage of Preprocessing

### Environment

node > 4.2.x

###Develop

```
npm run watch
```

###Commands

Release:

```
npm run release
```

Start local server:

```
npm run start:single
```

Start production server(need pm2):

```
npm run start
```
